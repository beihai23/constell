# Future · 暂不实现的扩展（不开 AC）

> 这些交互明确**不在当前 spec 范围**，只登记一行，避免遗漏；待优先级提升时升级为正式 feature 文件并开 AC。
> 原则：spec 与测试只覆盖「现有功能 + 明确在 scope 的缺失交互」。下列扩展不产 AC，以免测试套长期飘红、稀释信号。

## 消息

- **表情回应（reactions）**：消息上加减表情计数。
- **@提及 + 跳转**：文本中 @user/@role，触发通知与高亮。
- **消息撤回**：区别于删除，限时撤回并显示「消息已撤回」。
- **markdown / 格式工具栏**：粗体/斜体/代码块/引用。
- **消息固定（pin）**：频道置顶消息。

## 通知

- **通知中心 + 偏好设置**：桌面/邮件推送，按频道/DM 静音。
- **未读 @me 单独计数**。

## 社区 / 频道管理

- **社区设置**：改名、改描述、改图标、删社区。
- **频道设置**：改名、改主题、删频道、频道分类分组（category）。
- **私有社区 + 邀请加入**：当前仅公开社区可发现/自加入（见 [[community-search-discovery]]）。
- **频道级可见性/私有频道**。

## 成员 / 权限

- **角色与权限 UI**：自定义角色、position、权限位编辑（当前仅隐式 @everyone）。
- **成员资料完整卡**：头像、状态、活动、共同社区（部分见 `features/60` MEM-PROFILE-1）。
- **好友关系**：加好友/好友列表（区别于 DM）。
- **屏蔽用户（block/unblock）**：后端曾返回 501（见 [[e2e-test-report-2026-06-09]] 作弊 5）。

## 输入与可达性

- **拖拽 / 粘贴上传文件**。
- **输入中（typing）指示**：`features/50` 提为【目标-待建】，待后端 typing 通道。
- **键盘快捷键帮助（`?`）**。
- **无障碍审查（a11y）**：焦点管理、屏幕阅读器、对比度。

## 账户

- **密码强度指示器 + 「忘记密码」**（需后端）。
- **邮箱验证**。
- **「记住我」/ 会话管理**。
- **头像上传与个人资料编辑**。

## 平台

- **i18n 多语言**：当前全量硬编码英文。
- **响应式 / 移动端布局**：当前为桌面三栏固定。
- **浅色主题**：当前仅 Catppuccin Mocha 暗色。
